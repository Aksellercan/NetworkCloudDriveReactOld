# NetworkCloudDriveReactOld

Very basic frontend for NetworkCloudDrive API

## Setup
- **Create `.env` file in root directory and add `REACT_APP_API_URL = API_URL` otherwise won't work**

## Deploy
- Run in dev environment `npm start`
- Run Production builds using `npm run build` then `serve -s build` available on NPM [Serve](https://www.npmjs.com/package/serve)

## Incase of NPM issues
- Remove node modules `rm -rf node_modules`
- Reinstall them again `npm install`

*Probably won't be getting any UI enhancements anymore*
